# AI Connect — Claude Skills

A collection of open-source skills for Claude Code and Cowork by [Khalil Eid](https://aikhalileid.com).

## Available Skills

| Skill | Description |
|-------|-------------|
| [meta-ads](./meta-ads/) | Pull, analyze, manage, and create Meta ads (Facebook, Instagram, Messenger, Threads) via the Marketing API |

## How to Use

1. Download or clone this repo
2. Copy the skill folder into your Claude Code skills directory
3. Follow the setup guide inside each skill
